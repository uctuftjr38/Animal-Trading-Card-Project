My HTML Notes

Markup is the formatting of html text in and editor and tells the webpage what to do.

For example, emphasis tags are used to italicize and bold text.

Elements consist of opening/start tags and closing/end tags. Tags like break do not have closing tags and are called void/empty tags.

When nesting, make sure tags are used correctly. Paragraphs CAN NOT be nested inside each other!

Paragraphs create space or blocks around text that have height/width and top/bottom margins.

The div element is similar to the break element however it is used to organize and help lay out pages.

The paragraph and list elements have optional closing tags. However, some prefer to include closing tags to be more consistent in their code.

If you do decide to leave off the closing tags, the browser will know when to close that element once it sees the next opening tag.

Web addresses

A web address is called a URL.

URL stands for Uniform Resource Locator

UNIFORM = standardized
RESOURCE = a file or other data object
LOCATOR = address for finding something

In HTML, you must use the Fully Qualified or Absolute URL when referring to a webpage.

For example:

  • http://www.example.com (Unsecured)
  • https://www.example.com (Secured)
  • file://Users/MyName/notes.html (Local file)

Http, https, and file are Protocols or ways the browser can get something.

"://" separates protocol from Domain name (www.example.com).

Include a "/" at the end to link to the file path or leave empty if referring to the homepage of a site.

Links and the "a" tag

Example: a href="http://example.net"

  • "a" = anchor and requiqres a closing tag
  • "href=" = Attribute name (stands for hypertext reference)
  • "http://example.net" = Attribute Value or URL
  • Include descriptive Content after the opening tag to display link text in browser.

The DOCTYPE tag

Every HTML document must include (!DOCTYPE html) at the top of the page.

What does it do? It tells the browser the document should be treated as modern HTML.

Documents: Head and body

Head includes:

  • Title(required)
  • Stylesheets in CSS(fonts colors)
  • Scripts or programs that run in the browser(Ex. quizzes)
  • Metadata for search engines(keywords)

Body includes content like:

  • Paragraphs
  • Images
  • Links

The html tag goes after the document type and everything is enclosed within it(head and body).

The language type must also be included to show what language the document is written in.

CSS Vocabulary Review

  • DOM, or Document Object Model, is a tree structure with a node for each HTML element, piece of text, image, and any other object in the web page.
  • Selectors are one part of the CSS ruleset. They indicate which HTML element(s) the rule is for.
  • Declaration blocks are the other part of the CSS ruleset, they say how the rule will modify the elements indicated by selectors.
  • CSS rules are composed of a selector followed by a declaration block.
  • type is the simplest kind of selector. It indicates the name of one type of HTML element (such as "em" or "li")
  • class in HTML is an attribute that groups elements together that you want to have the same style.
  • id is an attribute that must be unique to a single element, used to identify it.

CSS defines the style and HTML applies the style to an HTML element.

Units

Pixel(px) is the basic unit of measurement in CSS. Pixels are what you'll see developers use to measure the size of a box element.
1 inch = 96 pixels
37.8 pixels = 1 centimeter

Point(pt) is a traditional typographical unit. Most common for font sizes.
72 pt = 1 inch

Other CSS units are relative to the size of something else on the page. For example, em is equal to the font size of the text in the element.

Boxes

CSS settings for boxes include:

  • Width
  • Height
  • Padding (space inside the boxes borders)
  • Border
  • Margin (space space outside the border)
